Pipeline

How It Works

From raw text to sentiment prediction — a clean NLP pipeline under the hood.

1

Text Preprocessing

Lowercase, strip HTML & URLs, remove punctuation, filter stopwords, apply Porter Stemming.

2

TF-IDF Vectorization

Convert cleaned text to a 5,000-dim sparse feature vector using bigram TF-IDF.

3

GRU Encoding

2-layer GRU with dropout encodes the feature vector into a dense 128-dim representation.

4

Classification

A fully connected layer + sigmoid outputs the probability of Positive sentiment.

Features

What Makes This Model Special

Built with best practices for production-ready NLP.

🧠

GRU Architecture

Upgraded from vanilla RNN to GRU — better long-range dependency capture and faster training.

🔄

BCEWithLogitsLoss

More numerically stable than BCELoss + sigmoid, preventing gradient vanishing during training.

📉

LR Scheduler + Clipping

StepLR decay and gradient clipping ensure stable convergence and prevent exploding gradients.

💾

Model Checkpointing

Saves the best model checkpoint automatically during training based on validation accuracy.

📊

Stratified Split

Ensures balanced class distribution in both train and test sets for unbiased evaluation.

🔍

Bigram TF-IDF

Captures two-word phrases like "not good" or "very bad" for richer text representation.
